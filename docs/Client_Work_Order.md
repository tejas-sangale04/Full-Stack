## 1. Project Overview

This Work Order authorizes the design, development, testing, and deployment of a **Full-Stack Restaurant Management System** — a modern, responsive web application built to digitize and streamline end-to-end restaurant operations. The system will serve customers, waitstaff, kitchen staff, and administrators through role-based dashboards and interfaces.

---

## 2. Scope of Work

### 2.1 Frontend Development
| Module | Description |
|---|---|
| **Landing Page** | Premium, responsive homepage with hero section, services showcase, multi-language support (English, Marathi, Hindi), and dynamic Open/Closed status indicator. |
| **Menu Page** | Interactive digital menu with category filtering, item details, and pricing. |
| **Table Reservation** | Online table booking system with date/time picker and confirmation flow. |
| **Offers & Rewards** | Customer-facing pages displaying current promotions, discounts, and loyalty rewards. |
| **Login / Authentication** | Secure login page with role-based access (Admin, Waiter, Kitchen Staff). |

### 2.2 Admin Dashboard
| Feature | Description |
|---|---|
| **Order Management** | View, track, and manage all customer orders in real-time. |
| **Menu Management** | CRUD operations for menu items, categories, and pricing. |
| **Reservation Management** | View and manage table bookings. |
| **Offer Management** | Create, update, and delete promotional offers. |
| **Revenue Tracking** | Dashboard with analytics, charts, and revenue summaries. |
| **Restaurant Status Control** | Toggle Open/Closed status displayed on the live website. |

### 2.3 Staff Dashboards
| Dashboard | Description |
|---|---|
| **Kitchen Dashboard** | Real-time order queue for kitchen staff with status updates. |
| **Waiter Dashboard** | Active orders and table assignments for waitstaff. |

### 2.4 Backend & Infrastructure
| Component | Description |
|---|---|
| **API Development** | RESTful API endpoints for all CRUD operations using Next.js API Routes. |
| **Database** | MongoDB database with Prisma ORM for data modeling and queries. |
| **Authentication** | NextAuth.js with credential-based authentication and role-based authorization. |
| **Security** | Password hashing (bcrypt), environment variable management, middleware-based route protection. |
| **Deployment** | Production deployment on Vercel with CI/CD pipeline. |
